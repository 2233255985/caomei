### 推理流程

```
POST /api/detect (图片)
  → detect_service.run_detect()
    → LocalDetector.detect(image_path)
      → YOLOv8SegEngine.detect(img)
        → ONNX Runtime session.run()
        → postprocess: decode → NMS → 返回 [{bbox, score}, ...]
    → 保存记录到 SQLite
  → 返回 {detections, count, avg_confidence, image_url}
```

### yolo_seg.py 要点

- `num_classes = 1`（硬编码，草莓只有 1 类）
- 后处理用 `mc[:, 7:]` 切片 mask 系数（当前未返回 mask，只返回 bbox + score）
- ONNX 输入: `(1, 3, 640, 640)` 归一化 float32
- ONNX 输出: concat `(1, 37, 8400)` + 3 个检测头 + mask_coeff `(1, 32, 8400)` + protos `(1, 32, 160, 160)`

### 部署流程（将 PC 模型部署到板子）

#### Step 1：PC 导出 ONNX

输出 `best.onnx`（~45MB, opset 11）。

#### Step 2：PC → 板子传文件

```powershell
# ONNX 模型
scp "C:\Users\tttjw\Desktop\code\atlas200\ultralytics-v8.0.45-relu6-tomato-seg\best.onnx" HwHiAiUser@192.168.1.2:/home/HwHiAiUser/

# 推理脚本（一次性更新）
scp "C:\Users\tttjw\Desktop\code\atlas200\ultralytics-v8.0.45-relu6-tomato-seg\xilinx\vart_py\yolo_acllite.py" HwHiAiUser@192.168.1.2:/home/HwHiAiUser/
scp "C:\Users\tttjw\Desktop\code\atlas200\ultralytics-v8.0.45-relu6-tomato-seg\xilinx\vart_py\server_acllite.py" HwHiAiUser@192.168.1.2:/home/HwHiAiUser/
```

#### Step 3：板子 ATC 转 OM

```bash
cd /home/HwHiAiUser
atc --model=best.onnx --framework=5 --output=best --soc_version=Ascend310 --input_shape="images:1,3,640,640" --output_type=FP32
```

输出 `best.om`（~27MB），耗时约 10-20 分钟。

#### Step 4：启动服务

```bash
python3.6 server_acllite.py
```

### OM 模型输出

| 索引 | 形状 | 说明 |
|------|------|------|
| 0 | (1, 37, 8400) | concat（box 4 + cls 1 + mask_coeff 32） |
| 1 | (1, 65, 80, 80) | stride 8 检测头 |
| 2 | (1, 65, 40, 40) | stride 16 检测头 |
| 3 | (1, 65, 20, 20) | stride 32 检测头 |
| 4 | (1, 32, 8400) | mask 系数 |
| 5 | (1, 32, 160, 160) | 原型 mask |

### yolo_acllite.py 后处理要点

- 从 `output[0]` concat 中切片：`mc = pred[:, 4 + num_classes:]`（nc=1 时为第 5 列起）
- `num_classes` 从 `classes.txt` 读取（当前 1 类：Strawberry）
- 生成 mask：`masks = sigmoid(dot(mc, protos.reshape(32, -1)))`
- mask 用 box 裁剪后 resize 到原图

## 五、常见问题

### 板子服务起不来

```bash
# 检查 Flask 是否安装
python3.6 -c "from flask import Flask; print('ok')"

# 没装的话 apt 安装
sudo -E apt install python3-flask -o Acquire::Check-Valid-Until=false
```

### 连接不上板子

```powershell
# 检查 RNDIS 网卡 IP
ipconfig
# 应看到 192.168.1.101

# 测试连通性
ping 192.168.1.2
```

### ATC 转换失败

```bash
# 确认 ONNX 文件完整
ls -lh best.onnx  # 应约 45MB

# 确认 ATC 可用
which atc

# 确认 CANN 版本
npu-smi info
```

### 后处理报 shapes not aligned

`classes.txt` 内容与模型 nc 不匹配。检查板子上 `model_data/classes.txt` 是否只有一行 `Strawberry`。
